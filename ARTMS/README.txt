This is the firmware for the StimSync device for Arduino Mega 2560. Oscilloscope and Keyboard mode are omitted.

ARTMS stands for Arduino Real Time Measurement System.

You will want to get the appropriate Teensyduino or Arduino software. Tools/Board should reflect your board type (Teensy3), Tools/SerialPort should specify the port where your device is attached and Tools USBType should be Serial+Keyboard. Once these settings are correct you can upload the firmware with File/Upload
